# AI週次勉強会 スライド制作計画

## 概要

| 項目 | 内容 |
|------|------|
| 対象 | CS（カスタマーサクセス）チーム、ビジネス職、〜10名 |
| AI知識レベル | ほぼゼロ（ChatGPT未使用〜触れたことある程度） |
| 使用ツール | Notion / Slack / Salesforce / スプレッドシート / Gemini |
| 端末環境 | Mac / Windows 混在 |
| 1回の時間 | 60分（ハンズオン重視） |
| 最終ゴール | Claude Code を最大限活用、他 AI との使い分けができる。業務の 30〜50% を AI 前提で構築し、「人がやる部分 / チャットAIと壁打ち / 任せる部分」を使い分けられるようになる |

## スライド生成ルール

### ファイル命名規則
```
scripts/session_NN.py   ← 生成スクリプト
output/session_NN.pptx  ← 出力ファイル
```

### 各スクリプトの構造
```python
"""Session NN: [タイトル]"""
from src.builder import SlideBuilder

b = SlideBuilder("output/sessionNN.pptx")
# スライドを順番に追加
b.save()
```

### デザインルール（DESIGN.md 準拠）
- Primary: `#EB9B4E`（アクセント・重要要素）
- Secondary: `#38516D`（見出し・ヘッダー）
- Neutral: `#F0F5ED`（背景・カード）
- Font Headline: `Plus Jakarta Sans`
- Font Body: `Work Sans`

### Codex への指示形式
各セッションを生成するときは、このファイルの該当セクションを読み込み、
`scripts/session_NN.py` を作成して `python3 scripts/session_NN.py` で実行すること。
